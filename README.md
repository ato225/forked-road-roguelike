# 갈림길 로그라이크

모바일과 데스크톱 브라우저에서 동작하는 선택지 기반 웹 로그라이크입니다.

## 실행

```powershell
cd C:\Users\user\side-scroller-demo
python -m http.server 8000
```

브라우저에서 `http://localhost:8000`을 열면 됩니다.

## 규칙

- 매 턴마다 `안전한 길`, `평범한 길`, `위험한 길` 중 하나를 선택합니다.
- 길마다 마을, 전투, 보물, 함정, 성소, 상인 등의 룸 확률이 다릅니다.
- 몬스터를 처치하면 골드를 얻고, 마을이나 상인에게서 소비 아이템을 살 수 있습니다.
- `치유 물약`, `화염 폭탄`, `방패 약병`은 인벤토리에 저장되며 상황에 맞게 사용할 수 있습니다.
- 보물과 정예 보상 장비는 실제 장착 UI 대신 영구 스탯 상승으로 적용됩니다.

## 포함 기능

- 플레이어/몬스터 기본 스탯
- 턴제 전투
- 마을 상점
- 인벤토리 사용
- 보물 장비 강화
- 로그 기록
- 로컬 스토리지 자동 저장
